# AI Coding Assistant

An AI-powered coding assistant developed using Flask and OpenAI API that helps users generate code, solve programming problems, and receive development guidance through natural language interactions.

The application provides an intuitive web interface where users can submit coding-related queries and receive AI-generated responses in real time, enhancing productivity and simplifying software development tasks.

## Features

- AI-powered code generation
- Programming problem solving
- Natural language query processing
- Real-time response generation
- REST API integration
- Responsive web interface
- Cross-Origin Resource Sharing (CORS) support

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask
- Flask-CORS

### AI Integration
- OpenAI API

## Workflow

1. User enters a coding-related query.
2. Flask backend processes the request.
3. Query is sent to the OpenAI API.
4. AI generates a relevant coding response.
5. Generated output is displayed on the web interface.

- Integrated Generative AI capabilities into a web application.
- Implemented API-based communication with large language models.
- Developed a scalable backend using Flask.
